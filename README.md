# LB-Web


## Installation Process

- [ ] [Laravel Documentation](https://laravel.com/docs/9.x/installation)

```
cp .env.example .env
php artisan key:generate
composer install

create database and update database creds in .env then run migration

php artisan migrate
php artisan db:seed

php artisan storage:link

php artisan serve
```
